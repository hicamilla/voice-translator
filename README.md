# Voice Translator
A mobile-friendly voice translation web app that lets you speak or type in one language and instantly get a translation in another. Built as a portfolio project using only vanilla HTML, CSS, and JavaScript.

## Features
- Select source and target languages from a dropdown
- Swap languages instantly with a single button
- Type text manually or use your microphone to speak
- Translate text using the MyMemory API
- Animated mic button with visual listening feedback
- Smooth fade-in animation on translation results
- Play the translated text out loud with one tap
- Status messages throughout to guide the user

## Technologies used
- HTML5, CSS3, vanilla JavaScript
- Web Speech API (speech to text)
- SpeechSynthesis API (text to speech)
- MyMemory Translation API (free, no API key required)

## How to use
1. Select your source language (the language you will speak or type in)
2. Select your target language (the language you want to translate to)
3. Type in the text area or tap the microphone button and speak
4. Tap Translate to get your translation
5. Tap the play button to hear the translated text out loud

## Notes
- Works best in Google Chrome
- Microphone permission is required for voice input
- The free tier of the MyMemory API supports up to 1000 words per day
- Built and tested as a portfolio/demo project
